# 🧮 Calculadora JavaScript

Pequeña aplicación web de calculadora desarrollada con **HTML5, CSS3 y JavaScript**.
Permite realizar operaciones matemáticas básicas de forma rápida y sencilla.

---

## 🚀 Funcionalidades

* ➕ Suma
* ➖ Resta
* ✖️ Multiplicación
* ➗ División
* 🧹 Botón para limpiar pantalla
* ⌫ Borrar último número (si lo añadiste)

---

## 🛠️ Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla JS)

---

## 📂 Estructura del proyecto

```
calculadora/
│── index.html
│── styles.css
│── script.js
```

---

## ▶️ Cómo usar el proyecto

1. Clona el repositorio:

```bash
git clone https://github.com/TU-USUARIO/TU-REPOSITORIO.git
```

2. Entra en la carpeta:

```bash
cd TU-REPOSITORIO
```

3. Abre el archivo `index.html` en tu navegador.

---

## 💡 Cómo funciona

La calculadora utiliza:

* Eventos `addEventListener`
* Manipulación del DOM
* Funciones para realizar operaciones matemáticas
* Control de errores (como división entre 0)
